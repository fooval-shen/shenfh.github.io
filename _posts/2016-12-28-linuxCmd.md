## 查看进程命令

```
ps
ps -e        //当前运行的所有进程
ps aux|grep 进程名/端口号

```

## 查看端口使用情况

```
netstat 
netstat -tln    // 查看已经使用的端口
netstat -tlnap  // 查看已经使用的端口和对应的程序
netstat -tnlp|grep 80
```
## 查看端口被哪个进程占用

```
fuser -n tcp 80
```

## 查看文件尾部的内容

```
tail -f filename // 监视filename文件的尾部内容（默认10行，增加参数 -n 10），刷新显示在屏幕上
```


## 获取内存大小

```
free -m 
free -m |grep "Mem" | awk '{print $2}'
```

## 查看磁盘使用情况

```
df -h

du -sh /路径
```

## 文件查找

```
find /路径 -name 关键字

eg:
find dev/ -name '*.c++'

find ./ -name "*.cpp"  | xargs grep 'aaa.c'   //在文件中查找关键字。
```


## 查看登录用户

```
w
```

## 查看登录日志

```
last
```

## 查看所有帐号的登录日志

```
lastlog
```
## 修改用户密码

```
sudo passwd 用户名
```

## 关机/重启

```
shutdown -h now
shutdown -r now  //重启
```

## 压缩文件

```
tar zcvf 压缩后文件.tar.gz  需要压缩的文件夹/
```

## 解压缩

```
tar zxvf 压缩文件.tar.gz
```

## 删除用户

```
userdel test

userdel -r test	//删除用户，同事删除/home目录下的test目录
```
